Please follow the installation instructions below to add those layouts to your OS.

Installation instructions:

1. Copy into the **Downloads** folder and unzip the **RuIntl_E.zip** (Latin layout) and **RuIntl_R.zip** (Cyrillic layout) archives and then run the appropriate **setup.exe** files to install **both** layouts;
2. Add the "Russian (international, ENG)" layout in the **English language** settings and the "Russian (international, RUS)" layout in the **Russian language** settings;
3. In the administrative language settings copy the "Current user" settings to the "Welcome screen" and to the "New user accounts" settings.
